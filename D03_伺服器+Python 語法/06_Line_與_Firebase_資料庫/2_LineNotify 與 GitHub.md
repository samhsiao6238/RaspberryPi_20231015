# 連動 Line Notify 與 GitHub 倉庫

_即時發送事件通知_

<br>

1. 進入 [官網](https://notify-bot.line.me/zh_TW/)。

   <img src="images/img_11.png" width=500px />

<br>

2. 滑動到可連動的服務，並選取 GitHub。

   <img src="images/img_10.png" width=500px />

<br>

3. 注意一下，這時的網址是在 `notify-bot.line.me` 之內，而非 `GitHub`。這與另一個服務 `IFTTT` 的設置不同， `IFTTT` 是轉入該服務的平台中進行後續步驟的設定。

   ![](images/img_14.png)

<br>

4. 授權。

   <img src="images/img_01.png" width=400px />

<br>

5. 驗證：務必建立雙重驗證。

   ![](images/img_02.png)

<br>

6. 可參考官網 [說明文件](https://docs.github.com/en/webhooks)。

   <img src="images/img_03.png" width=400px />

<br>

7. 選擇要接收通知的帳號，在這裡就是自己的 Line ID；並選取連動的 Repo 。

   <img src="images/img_04.png" width=500px />

<br>

8. 選擇要接收通知的聊天室。

   ![](images/img_05.png)

<br>

9. 同意連動。

   ![](images/img_06.png)

<br>

10. 完成連動。

      ![](images/img_07.png)

<br>

11. 在該通知尚未加入所選群組之前，舉凡在群組內的成員都會收到需將該通知加入群組的訊息。

      <img src="images/img_40.png" width=300px />

<br>

12. 進入 `Line App` 把 `Line Notify` 加入群組中。

      <img src="images/img_08.png" width=400px />

<br>

13. 邀請加入。

      ![](images/img_12.png)

<br>

14. 同步 Github Repo 時發送通知到群組中。

      <img src="images/img_13.png" width=300px />

<br>

___

_END：以上建立簡易的通知_
