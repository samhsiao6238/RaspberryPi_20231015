# 虛實互動

_偵測食指指尖與指定區域的觸碰_

<br>

## 說明

1. 這個範例透過 OpenCV 的 `cv2.flip()` 函數進行畫面水平翻轉，讓影像看起來像是照鏡子。

<br>

2. 增加獲取攝像頭影像比例，然後指定寬度，並讓高度同比例變動。

<br>

3. 封裝觸碰偵測的功能。

    ```python
    def check_if_touched(hand_landmarks, touch_area, width, height):
        # 取得食指末端 x, y 座標
        finger_X = hand_landmarks.landmark[7].x * width
        finger_Y = hand_landmarks.landmark[7].y * height

        # 檢查是否觸碰
        if (touch_area[0] < finger_X < touch_area[0] + 80) and (touch_area[1] < finger_Y < touch_area[1] + 80):
            return True
        return False
    ```

<br>

## 範例

1. 添加一個全局變數 `is_Touched` 來判斷手掌位置是否與設定的區域重疊。

<br>

2. 完整程式碼。

    ```python   
    '''
    這是一個動態的腳本，畫面中央顯示一個紅色框
    手勢觸碰可觸發邏輯
    '''

    import cv2
    import mediapipe as mp
    import random

    # 繪圖方法
    mp_drawing = mp.solutions.drawing_utils
    # 繪圖樣式
    mp_drawing_styles = mp.solutions.drawing_styles
    # 初始化手掌偵測對象
    mp_hands = mp.solutions.hands
    # 初始化攝像頭
    cap = cv2.VideoCapture(0)

    # 獲取攝像頭原始分辨率
    original_width = cap.get(cv2.CAP_PROP_FRAME_WIDTH)
    original_height = cap.get(cv2.CAP_PROP_FRAME_HEIGHT)
    # 計算比率
    aspect_ratio = original_height / original_width
    # 設定指定寬度
    new_width = 600
    # 依據寬度設定高度
    new_height = int(new_width * aspect_ratio)


    # 自定義觸碰偵測函數
    def check_if_touched(hand_landmarks, touch_area, width, height):
        finger_X = hand_landmarks.landmark[7].x * width
        finger_Y = hand_landmarks.landmark[7].y * height
        if touch_area[0] < finger_X < touch_area[0] + 80 and touch_area[1] < finger_Y < touch_area[1] + 80:
            return True
        return False

    # mediapipe 啟用偵測手掌
    with mp_hands.Hands(
        model_complexity=0,
        min_detection_confidence=0.5,
        min_tracking_confidence=0.5) as hands:

        if not cap.isOpened():
            print("無法找到攝像頭")
            exit()

        # 預設觸碰區域
        touch_area = [0, 0]
        
        # 設定是否觸碰狀態為 True
        is_Touched = True
        while True:
            success, img = cap.read()
            
            if not success:
                print("無法獲取畫面")
                break
            # 水平翻轉
            img = cv2.flip(img, 1)
            
            # 調整畫面尺寸(寬度, 高度)
            img = cv2.resize(img,(new_width,new_height))
            # 取得攝影機影像尺寸(高度, 寬度, 通道數)
            size = img.shape   
            # 取得畫面寬度、高度
            _width, _height = img.shape[1], img.shape[0]

            if is_Touched:
                # 如果沒有碰到，就一直是 False ( 不會更換位置 )
                is_Touched = False
                touch_area = [random.randint(50, _width-50), random.randint(50, _height-100)]
            
            # 將 BGR 轉換成 RGB
            img2 = cv2.cvtColor(img, cv2.COLOR_BGR2RGB)
            # 偵測手掌
            results = hands.process(img2)
            
            if results.multi_hand_landmarks:
                for hand_landmarks in results.multi_hand_landmarks:
                    if check_if_touched(hand_landmarks, touch_area, _width, _height):
                        is_Touched = True
                    # 將節點和骨架繪製到影像中
                    mp_drawing.draw_landmarks(
                        img,
                        hand_landmarks,
                        mp_hands.HAND_CONNECTIONS,
                        mp_drawing_styles.get_default_hand_landmarks_style(),
                        mp_drawing_styles.get_default_hand_connections_style())

            # 畫出觸碰區
            cv2.rectangle(
                img, 
                (touch_area[0], touch_area[1]),
                (touch_area[0] + 80, touch_area[1] + 80),
                (0, 0, 255),
                -1   # 線條粗細設置為 -1 代表填滿
            )

            # 顯示影像並設置標題
            cv2.imshow('Gesture_Touch', img)
            # 檢查是否有按下'ESC'、'q'鍵或關閉視窗
            key = cv2.waitKey(1) & 0xFF
            if (
                key == 27
                or key == ord("q")
                or cv2.getWindowProperty("Gesture_Touch", cv2.WND_PROP_VISIBLE) < 1
            ):
                break

    cap.release()
    cv2.destroyAllWindows()
    ```
