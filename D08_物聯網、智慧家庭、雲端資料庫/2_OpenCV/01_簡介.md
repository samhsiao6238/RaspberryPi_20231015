# OpenCV

<br>

## 準備工作

1. 建立虛擬環境。

    ```bash
    python -m venv envOpenCV
    ```

<br>

2. 更新。

    ```bash
    sudo apt update && sudo apt upgrade -y
    ```

<br>

3. 安裝套件 `opencv-python` 。

    ```bash
    pip install opencv-python
    ```

<br>

4. 驗證 `opencv-python` 確實安裝完成，直接在終端機中運行指令即可。

    ```bash
    python -c "import cv2; print(cv2.__version__)"
    ```

    ![](images/img_01.png)

<br>
