# Linux 指令

<br>

## 連線相關指令

1. `ssh`

    ```bash
    # 連線遠程主機
    ssh sam6238@raspi-2023-05
    # 指定端口
    ssh -p 2222 sam6238@raspi-2023-05
    # 指定私鑰檔案
    ssh -i ~/.ssh/mykey.pem sam6238@raspi-2023-05
    # 提供不同級別的調試訊息
    ssh -vvv sam6238@raspi-2023-05
    ```
    _若要更改端口，必須編輯 `/etc/ssh/sshd_config` 文件_
    ```bash
    # 編輯文件，將 Port 改為指定端口號即可
    # 服務檔案修改後要重啟 `sudo systemctl restart ssh`
    sudo nano /etc/ssh/sshd_config
    ```

<br>

2. `ping`

    ```bash
    # 測試與遠端伺服器的連線狀態，輸出的 time 代表往返時間ms
    ping 0.0.0.0
    ```

<br>

3. `ifconfig`

    ```bash
    # 顯示網路介面的資訊
    ifconfig
    # 顯示所有網路介面，包括停用的
    ifconfig -a
    # 顯示特定網路介面的資訊
    ifconfig eth0
    ```

<br>

4. `netstat`

    ```bash
    # 顯示網路連線、路由表、端口統計等網路資訊
    netstat -tuln
    # 顯示所有端口和監聽的連線
    netstat -a
    # 顯示TCP和UDP的連接，並且顯示端口號而不是服務名
    netstat -tuln
    ```

<br>

5. `traceroute`

    ```bash
    # 顯示數據包從本地到達目的地的路由
    traceroute www.example.com
    ```

<br>

6. `nslookup`

    ```bash
    # 查詢域名系統 (DNS) 的域名或IP地址
    nslookup www.example.com
    ```

<br>

7. `dig`

    ```bash
    # DNS 查詢工具，用於取得域名對應的 IP 地址等 DNS 記錄
    dig www.example.com
    ```

<br>

## 一般指令

1. `reboot`

    ```bash
    # 重啟啟動
    reboot now
    ```

<br>

2. `shutdown`

    ```bash
    # 關機
    shutdown -h now
    ```

<br>

3. `sudo`

    ```bash
    # 以 root 權限執行指令
    sudo apt update
    ```

<br>

4. `apt`

    ```bash
    # Debian 套件管理器的指令
    sudo apt install <套件>
    ```

<br>

## 文件和目錄操作

1. `ls`

    ```bash
    # 列出目錄檔案
    ls -l
    ls -al
    ```

<br>

2. `pwd`

    ```bash
    # 顯示當前路徑
    pwd
    ```

<br>

3. `cd`

    ```bash
    # 切換目錄
    cd /home/sam6238
    # 切換到家目錄
    cd ~
    ```

<br>

4. `touch`

    ```bash
    # 建立檔案
    touch test.txt
    ```

<br>

5. `nano`

    ```bash
    # 編輯檔案
    nano test.txt
    ```

<br>

6. `cat`

    ```bash
    # 串連檔案內容
    cat file1.txt file2.txt > combined.txt
    # 顯示檔案內容
    cat test.txt
    # 建立文件
    cat > newfile.txt
    # 在文件末尾追加內容
    cat >> filename.txt
    ```

<br>

7. `mv`

    ```bash
    # 移動檔案
    mv test.txt test1.txt
    ```

<br>

8. `cp`

    ```bash
    # 複製檔案
    cp test1.txt test2.txt
    ```

<br>

9. `mkdir`

    ```bash
    # 建立目錄
    mkdir test
    ```

<br>

10. `rm`

    ```bash
    # 刪除檔案或目錄
    rm -rf <檔案或目錄>
    ```

<br>

11. `rmdir`

    ```bash
    # 刪除目錄
    rmdir <目錄>
    ```

<br>

12. `scp`

    ```bash
    # 透過ssh複製檔案到遠端伺服器
    scp <本地檔案> sam6238@0.0.0.0:<遠端檔案名稱>
    ```

<br>

13. `find`

    ```bash
    # 在文件系統中搜索和定位文件和目錄
    find / -name test.txt
    ```

<br>

14. `grep`

    ```bash
    # 在檔案中搜索特定的文字
    grep "test" filename.txt
    ```

<br>

15. `less`

    ```bash
    # 用於分頁查看大檔案的內容
    less largefile.txt
    ```

<br>

16. `more`

    ```bash
    # 用於分頁查看大檔案的內容
    more largefile.txt
    ```

<br>

17. `tree`

    ```bash
    # 顯示當前資料夾
    tree -L 1
    ```

<br>

## 權限指令

1. `chmod`

    ```bash
    # 更改檔案權限
    chmod 755 <檔案>
    ```

<br>

2. `chown`

    ```bash
    # 更改檔案擁有者
    chown <帳號>:<群組> <檔案>
    ```

<br>

## 下載與解壓縮

1. `wget`

    ```bash
    # 下載網路上的檔案
    wget <檔案URL>
    ```

<br>

2. `tar`

    ```bash
    # 打包或解壓縮
    tar -xzvf <檔案名稱.gz>
    # tar 參數說明
    # z: gzip壓縮
    # x: 提取文件
    # v: 顯示詳細輸出
    # f: 指定要操作的文件
    ```

<br>
